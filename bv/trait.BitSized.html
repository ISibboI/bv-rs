pub trait BitSized {
    type Block: BlockType;
    fn bit_len(&self) -> u64;

    fn block_len(&self) -> usize { ... }
}

Interface for a sequence of bits that knows its length.

All bit-sized things are made out of blocks, which are typically unsigned primitive integer types such as u8 or usize. Thus, they also know their length in blocks.

This trait exists mainly to be a common supertrait of Bits and BitSliceable.

Associated Types

type Block: BlockType

Required Methods

Provided Methods

The length of the vector in blocks.
